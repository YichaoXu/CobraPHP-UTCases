# Pattern: Recursion

## Category

Function

## Definition

One of the way a function can call itself (recursion) is through _arguments.callee_.

## Instances

### Instance 1

- CATEGORY: D1
- FEATURE vs INTERNAL API: FEATURE
- INPUT SANITIZERS: NO
- SOURCES AND SINKS: NO
- NEGATIVE TEST CASES: NO
- CODE:

```javascript
var b = 0;
function rec(val){
    if(b === 0){
        b = 1; 
        arguments.callee(val);
    }else{
        res.write(val);
    }
}
```
